# Exercise 1 - Control Structures

## Scenario 1
Apply a 1% reduction in loan interest rates for customers above 60 years of age.

## Scenario 2
Promote customers with balances greater than 10000 to VIP status.

## Scenario 3
Generate reminder messages for loans due within the next 30 days.
