# Exercise 7 - Financial Forecasting

## Objective
Predict future financial values using a recursive approach.

## Concept Used
Recursion is a technique where a method calls itself until a base condition is reached.

## Formula

Future Value = Present Value × (1 + Growth Rate)^Years

## Time Complexity
O(n)

## Optimization Techniques
- Memoization
- Dynamic Programming

## Conclusion
Recursion offers a simple solution for forecasting future values based on a fixed growth rate.
