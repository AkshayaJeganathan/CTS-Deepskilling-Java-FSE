# Exercise 2 - E-Commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search algorithms to search products efficiently.

## Algorithms Used

### Linear Search
Checks each product one by one until the target product is found.

Time Complexity:
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
Divides the search space into two halves repeatedly.

Time Complexity:
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Conclusion
Binary Search provides better performance for large datasets when data is sorted.
