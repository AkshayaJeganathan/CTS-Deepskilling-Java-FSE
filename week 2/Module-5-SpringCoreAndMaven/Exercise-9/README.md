# Exercise 9 : Creating a Spring Boot Application

## Scenario
Develop a Spring Boot application for the Library Management System to simplify configuration and deployment.

## Objective
Implement a Spring Boot application with REST APIs for performing CRUD operations on books.

## Features Implemented
- Spring Boot application setup using Spring Initializr.
- Added dependencies:
  - Spring Web
  - Spring Data JPA
  - H2 Database
- Configured database properties in application.properties.
- Created Book entity.
- Created BookRepository interface.
- Developed BookController for handling REST endpoints.
- Implemented CRUD operations for books.
- Successfully executed and tested the application.

## Project Name
LibraryManagementBoot

## Main Components
- Book.java
- BookRepository.java
- BookController.java
- application.properties
- LibraryManagementBootApplication.java

## Technologies Used
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Maven

## Outcome
Successfully created and executed a Spring Boot application for Library Management with RESTful CRUD operations.
